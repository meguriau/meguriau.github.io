2020年11月12日

时间久了会发现，当你有小情绪的时候你就会沉默……虽然你每次都会说我沉默，但我感觉你的次数明显比我的多……现在视频的时候就看到，平时微信沉默的时候的你是怎样的样子了……

我又怎么会介意呢．你问我的那个问题过于假设，于是我就拿了你之前说的来说了．我心里想，大概我这样亲口跟你说一次，可以给你信心吧．没想到起了反效果……

我一直都觉得，因为你没什么自信，也容易自责，所以要让你清楚明白我的想法这件事很重要，一点也不能含糊，也不能有丝毫误解．我想了一天，也还是没想起来昨晚哪里话让你听成了你听到的那句了，可能不是同一个上下文，也可能不是这个句子．

我仔细想了想，会让我“有想法”的瞬间，就是7月5日跟上两周．除此之外，我一直在拼命奔向你．是的，相比现在你的，我觉得我还可以努力付出一点．在我看来，你或许没有信心，但已经比之前好好多了，我对你有，你不也默默地努力着．你大概会觉得我这样单方面会累吧……我想我应该也会．但像我之前跟你说的一样，如果有那么一刻让你看到，我想你那个时候可以摸摸我的头，抱抱我，抱紧我．

2020年11月08日

大夫山，没有很高，走在山路上，像走在农村里的小路一样，并不感觉像在爬山．偶尔能看见一两个小“水塘”，也能看到一丛丛的芦苇．在小路上，因为骑车的人比较多，所以得小心．有从身后突然出现，也有迎面骑过来，应该听到的不是提醒行人的响铃声，而是刹车声“JI——”，极其刺耳，而且一路上都能听到．这很让我抓狂．

我对它的印象不是很好．

看到你依旧还会害羞的样子，就知道你跟别人不一样，至少在我心里，你是独一无二的．我会努力去珍惜和守护好这一份纯真．

被你说了．回来我想了想，是不是我做得不一样了．是不是自己没有察觉表现出来的和自己心里所认为的自己是不一样的，理论上，我自己应该是知道的……但今天不知道是不是自己不知道．想了想，是不是最近受你影响，有点变得小心，怕让你感受到压力和不安；另外，自从你说会想自己消化事情之后，我是不是变得会越来越少过问．不确定，但最后给了自己这两个理由来让自己入睡．但我想让我的女孩知道，我从没对她冷漠，我也一直在．如果方式不对，甚至我自己都不知道的情况下，我也不放弃努力寻找合适的方式．

2020年11月07日

上学的时候一直都不是班里长得最高的，但一直是长得第二高的．班里最高的男生都总是打篮球的大佬，我也觉得他好厉害．我初中的时候，中学吃完饭就会和另外3个同学去打一中午的篮球，直到下午上课．也就只有我们几个不休息的．我对篮球的兴趣一直止步于投篮．虽然就这样的过了好几年，但兴趣一直提不起来．另外几个和我一起玩的同学似乎也是这样．

要说我没有集体意识不喜欢集体活动的球类么，好像也不是，我对排球的兴趣就远远大于篮球，虽然也不怎么会打．而事实上，班里的同学就会觉得，你长这么高，怎么不去打篮球呢．似乎长得高就要去打篮球是理所当然的．的确，作为硬件条件，长得高很有优势．虽然我心里不是很爽，但慢慢也从初中、高中和大学习惯了过来：“你长那么高，怎么不去打篮球呢？”

  • 我不是很会打……我不喜欢打……我不打篮球……
  • 关你屁事呢（心想，其实也没到这程度）

我一直觉得能力越大，责任越大．以前我以为跟“你长那么高，怎么不去打篮球呢？”是一样的．慢慢地发现，这不太一样．以能力来说，还可以稍微绑架一下别人；但兴趣，不可以．
